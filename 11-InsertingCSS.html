Inline Cascading Style Sheets

We can apply CSS in a single element by inline CSS technique.
Inline means in the after the opening tag before closing angle bracket.
Example, <p style="font-family: verdana">Hello<p>

Internal Cascading Style Sheets

An internal CSS is used to define a style for a single HTML page.
Internal CSS can be applied to multiple elements.
Internal CSS is defined in the <head> section of HTML page, inside <style> element.

External Cascading Style Sheets

External style sheet is used for defining many HTML pages.
Link of the external style sheet should be added in the head section of each pages where it is needed.
